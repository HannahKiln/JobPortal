Introduction To Software Engineering Course Work Job Portal Application

This project has been developed to allow users to find employment outside of their country of residence. 

Getting Started
- Download the zip file
- Extract it
- Open NetBeans
- On the left hand side select the services tab
- Expand the databases section
- Right click on “Java DB” and select “start server”
- Navigate to where you extracted the program
- Open the “dist” folder
- Double click on the “JobPortal” executable file.  

Prerequisites
This application requires the IDE NetBeans to function.
